## Day 1 | 9月28日（周一）基督城 → 蒂卡普湖

### 行程概览
基督城机场抵达 → 取车 → 简单逛市区 → 驱车前往蒂卡普湖

### 详细时间安排

| 时间 | 活动 | 备注 |
|------|------|------|
| **06:10** | 航班抵达基督城机场 | 奥克兰飞来的国内航班 |
| **06:10-08:00** | 出关、取行李 | 国内航班很快，约1-1.5小时 |
| **08:00-09:00** | 前往租车柜台取车 | 机场内或附近租车中心 |
| **09:00-10:00** | 取车手续、检查车辆 | 拍照记录车身状况 |
| **10:00-10:30** | 超市采购 | Countdown或New World，买零食、水、早餐 |
| **10:30-11:30** | 基督城市区简单逛逛 | 纸板教堂（Cardboard Cathedral）、植物园外围 |
| **11:30-12:00** | 午餐 | 市区随便吃点，或路上解决 |
| **12:00-15:00** | **驱车前往蒂卡普湖** | 约227km，3小时，走SH1转SH8 |
| **15:00-15:30** | 入住酒店 | Peppers Bluewater Resort |
| **15:30-17:30** | 湖边散步、好牧羊人教堂 | 拍照、发呆、看湖 |
| **17:30-18:30** | 湖边等日落 | 9月底日落约18:30-19:00 |
| **18:30-19:30** | 晚餐 | 酒店餐厅或镇上 |
| **19:30-21:00** | 休息、适应时差 | 早点睡，明天早起 |

### 今日亮点
- 🌟 第一次近距离看蒂卡普湖的牛奶蓝色
- 🌟 好牧羊人教堂打卡
- 🌟 湖边散步，适应新西兰的纯净空气

### 驾驶信息
| 项目 | 详情 |
|------|------|
| 路线 | Christchurch → Ashburton → Fairlie → Lake Tekapo |
| 距离 | 约227km |
| 时间 | 约3小时 |
| 路况 | 全程铺装公路，路况好 |
| 加油站 | Ashburton或Fairlie可加油 |

### 餐饮推荐
| 餐次 | 推荐 | 备注 |
|------|------|------|
| 午餐 | 基督城市区或路上 | 简单解决 |
| 晚餐 | Mackenzies Cafe Bar & Grill | 蒂卡普湖镇上，牛排不错 |

### 注意事项
- 今天主要是赶路+适应，不要太累
- 基督城到蒂卡普湖沿途风景一般，专心开车
- 到了蒂卡普湖先办入住，再慢慢逛
- 晚上早点休息，明天要早起去库克山

| **15:00-17:00** | 酒店休息/湖边发呆 | 补觉或游泳（酒店有泳池）|

| **23:00** | 回酒店休息 | |

- 🌟 库克山Hooker Valley步道，雪山冰川近在咫尺
- 🌟 普卡基湖牛奶蓝，新西兰最美湖泊之一
- 🌟 蒂卡普湖暗夜星空，肉眼可见银河

### 驾驶信息
| 项目 | 详情 |
|------|------|
| 路线 | Lake Tekapo → Mount Cook → Lake Pukaki → Lake Tekapo |
| 距离 | 约210km往返 |
| 时间 | 约3小时驾驶 |
| 路况 | 铺装公路，风景极美 |

### 步道信息：Hooker Valley Track
| 项目 | 详情 |
|------|------|
| 全程往返 | 10km，约3小时 |
| 你们走到的第一座吊桥 | 约2.5km，1小时往返 |

| 风景 | 吊桥、冰川湖、库克山全景 |

### 观星指南
| 项目 | 详情 |
|------|------|
| 最佳时间 | 22:00-23:30（月亮23:30升起）|
| 最佳地点 | 好牧羊人教堂旁湖边 |
| 温度 | 约0-5°C，非常冷 |
| 必备 | 厚外套、手套、帽子、三脚架（拍照）|

### 餐饮推荐
| 餐次 | 推荐 | 备注 |
|------|------|------|
| 早餐 | 酒店早餐 | |
| 午餐 | 自带三明治/ Twizel镇 | 库克山附近餐厅少且贵 |
| 晚餐 | Kohan Restaurant | 蒂卡普湖镇上，日式料理，口碑好 |

- 库克山天气多变，带雨具和保暖衣物
- 步道没有厕所，出发前解决
- 观星时不要用白光手电，影响他人
- 如果天气不好 cloudy，观星效果大打折扣
