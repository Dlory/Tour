## 一、蒂卡普湖（Lake Tekapo）— 2晚

### 🌟 强烈推荐：Peppers Bluewater Resort

| 项目 | 详情 |
|------|------|
| **地址** | State Highway 8, Lake Tekapo |
| **房型推荐** | Lake View Room（湖景房）或 1-Bedroom Suite |
| **价格** | NZD 250-450/晚 |
| **预订** | https://www.peppers.co.nz/bluewater |
| **电话** | +64 3 680 2888 |

**为什么推荐：**
- 蒂卡普湖最好的酒店之一，正对湖面
- 湖景房阳台即可看湖+好牧羊人教堂
- 晚上步行5分钟到湖边观星
- 有室内恒温泳池，春天游泳很舒服
- 房间带厨房，可以自己做饭

**预订技巧：**
- 湖景房比普通房贵 NZD 50-100，但值得
- 提前3个月预订，旺季很容易满房
- Booking.com 上有时有"Genius"折扣

## 六、预订渠道推荐

### 国际平台

| 平台 | 网址 | 特点 |
|------|------|------|
| **Booking.com** | booking.com | 选择最多，Genius会员折扣，免费取消 |
| **Agoda** | agoda.com | 亚洲用户友好，有时比Booking便宜 |
| **Expedia** | expedia.com | 套餐优惠（机票+酒店）|

### 酒店官网

| 优势 | 劣势 |
|------|------|
| 有时有独家优惠 | 需一家家查 |
| 积分/会员福利 | 没有比价 |
| 取消政策更灵活 | |

### 我的建议

1. **先在 Booking.com 搜索** → 看价格、评价、取消政策
2. **对比酒店官网价格** → 有时官网更便宜或有额外福利
3. **优先选"免费取消"** → 计划有变可以退
4. **提前2-3个月预订** → 旺季好酒店很快满房

## 七、预订技巧

### 省钱技巧

1. **Booking.com Genius会员**
   - 住满2次自动升级Genius Level 1，有10%折扣
   - 住满5次升级Level 2，有15%折扣+免费早餐
   - 如果已有账号，登录后搜索可能更便宜

2. **灵活日期**
   - 如果前后差1-2天价格差很多，可以考虑微调行程

3. **直接联系酒店**
   - 对于小型精品酒店，发邮件直接问可能有折扣
   - 特别是住2晚以上，可以问是否有"stay 2 nights, save 10%"

### 注意事项

1. **确认取消政策**
   - 选"免费取消"的，以防行程变动
   - 注意免费取消的截止时间（通常是入住前24-48小时）

2. **确认含早餐**
   - 有些价格含早餐，有些不含
   - 新西兰酒店早餐通常 NZD 25-40/人，提前确认

3. **确认停车**
   - 自驾需要停车，确认酒店是否免费停车
   - 皇后镇有些酒店停车要额外收费 NZD 10-20/天

4. **特殊要求**
   - 蜜月/纪念日可以备注，酒店可能送香槟或升级
   - 需要高楼层/安静房间，预订时备注
