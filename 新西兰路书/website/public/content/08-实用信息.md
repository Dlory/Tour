## 一、电话卡/网络

### 推荐方案：落地买当地SIM卡

新西兰主要有三大运营商：

| 运营商 | 优点 | 缺点 | 推荐套餐 |
|--------|------|------|----------|
| **Spark** | 覆盖最好，偏远地区有信号 | 价格稍贵 | NZD 49 / 10GB / 30天 |
| **Vodafone** | 网速快，城市覆盖好 | 山区信号弱 | NZD 45 / 10GB / 30天 |
| **2degrees** | 最便宜 | 覆盖一般 | NZD 35 / 10GB / 30天 |

### 购买地点

| 地点 | 说明 |
|------|------|
| **基督城机场** | 到达大厅有柜台，最方便 |
| **超市/便利店** | Countdown、New World有售 |
| **运营商门店** | 市区有Spark/Vodafone门店 |

### 推荐购买流程

1. 基督城机场到达大厅找 **Spark** 或 **Vodafone** 柜台
2. 出示护照（实名制）
3. 选择套餐：推荐 **NZD 49-68 的月套餐**
   - 包含：10-25GB 流量 + 本地通话 + 短信
   - 有效期：30天
4. 工作人员帮忙安装、激活
5. 测试网络

> 汇率波动，出行前再查。建议下载 **XE Currency** APP实时查看。

### 支付方式

| 方式 | 普及度 | 建议 |
|------|--------|------|
| **信用卡（Visa/Mastercard）** | ⭐⭐⭐ 最普遍 | **首选**， everywhere |
| **借记卡** | ⭐⭐⭐ 普遍 | 可用，但可能有手续费 |
| **现金** | ⭐⭐ 部分地方 | 备用，小费、集市 |
| **支付宝/微信** | ⭐ 少数地方 | 部分中国游客多的店 |
| **银联** | ⭐⭐ 部分地方 | 大商场、免税店可能可用 |

### 银行卡建议

1. **带2张信用卡** — Visa + Mastercard，一张备用
2. **通知银行出国** — 避免刷卡被冻结
3. **确认无境外手续费** — 部分银行收1-3%
4. **带少量现金** — NZD 200-300 备用

### ATM取现

| 项目 | 说明 |
|------|------|
| 随处可见 | 银行、超市、加油站都有ATM |
| 手续费 | 每笔约 NZD 3-5 + 国内银行手续费 |
| 建议 | 一次取够，减少手续费 |

## 五、入境注意事项

### 新西兰入境检疫（非常严格！）

新西兰对入境物品检查极严，**违规罚款 NZD 400 起**。

#### ❌ 禁止携带

| 物品 | 说明 |
|------|------|
| 新鲜水果/蔬菜 | 任何种类 |
| 肉类/蛋类 | 任何种类，包括方便面里的肉包 |
| 蜂蜜 | 禁止 |
| 种子/植物 | 禁止 |
| 泥土/沾泥土的鞋子 | 需申报并清洁 |
| 宠物食品 | 禁止 |

#### ⚠️ 必须申报

| 物品 | 说明 |
|------|------|
| 药品 | 处方药需带医生证明 |
| 茶叶 | 可带，需申报 |
| 干货 | 如木耳、香菇等，需申报 |
| 户外装备 | 帐篷、登山鞋等（检查泥土）|
| 香烟 | 限50支，超量需缴税 |
| 酒类 | 限3瓶（每瓶不超过1.12升）|

#### ✅ 可以携带

| 物品 | 说明 |
|------|------|
| 巧克力/糖果 | 可以 |
| 饼干/零食（无肉）| 可以 |
| 方便面（无肉包）| 可以 |
| 化妆品/护肤品 | 可以 |
| 个人衣物 | 可以 |

### 入境流程

1. **填写入境卡** — 飞机上发，英文填写
2. **护照检查** — 自助通关（eGate）或人工
3. **取行李** — 转盘取托运行李
4. **生物安全检查** — 行李过X光，可能开箱检查
5. **入境完成** — 出关

### 入境卡填写要点

- **所有食品必须申报** — 不确定就勾"Yes"
- **户外装备必须申报** — 登山鞋、帐篷等
- **药品必须申报** — 特别是处方药
- **诚实申报** — 隐瞒被发现罚款更重

### 小费文化

| 场景 | 是否给小费 |
|------|-----------|
| 餐厅 | 不强制，服务特别好可给5-10% |
| 酒店 | 不强制 |

**新西兰不强制小费**，服务质量不因小费而变。

### 饮水

| 项目 | 说明 |
|------|------|
| 自来水 | **可以直接喝**，非常干净 |
| 瓶装水 | 超市买，NZD 1-3 |

| 地点 | 情况 |
|------|------|

| 超市/商场 | 免费 |

- 酒店/公共场所通常有分类垃圾桶

## 十一、重要提醒

1. **右舵左行** — 上车先适应，出停车场特别注意
2. **安全带必系** — 全员，违者罚款 NZD 150
3. **超速罚款重** — 摄像头多，严格遵守限速
4. **入境申报诚实** — 不确定就勾Yes，隐瞒罚款更重
5. **天气变化快** — 山区可能突然下雨，带冲锋衣
6. **紫外线极强** — 防晒霜+墨镜，即使阴天也要涂
7. **尊重自然** — 不喂野生动物，不留垃圾，走指定步道
