# 06 - 穿衣与装备清单

> 时间：9月底-10月初（新西兰春季）
> 地区：南岛（蒂卡普湖、库克山、瓦纳卡、皇后镇、奥马鲁）
> 气温：白天 10-18°C，夜间/山区 0-8°C

---

## 一、天气概况

### 9月底-10月初南岛天气

| 地点 | 白天温度 | 夜间温度 | 天气特点 |
|------|----------|----------|----------|
| 蒂卡普湖 | 12-18°C | 0-5°C | 干燥，昼夜温差大 |
| 库克山 | 8-15°C | -2-3°C | 山区天气多变，可能下雪 |
| 瓦纳卡 | 12-18°C | 2-6°C | 温和，偶尔降雨 |
| 皇后镇 | 10-16°C | 2-6°C | 多变，可能突然下雨 |
| 奥马鲁 | 12-17°C | 5-8°C | 沿海，较温和 |

### 穿衣原则：洋葱式穿搭

新西兰春季天气多变，一天可能经历四季。建议**多层穿搭**，方便随时增减。

```
内层（排汗）→ 中层（保暖）→ 外层（防风防雨）
```

## 三、按场景穿搭建议

### 场景1：日常城市/小镇漫步

```
速干长袖T恤 + 薄抓绒衣 + 牛仔裤 + 休闲鞋 + 墨镜
```

### 场景2：库克山Hooker Valley步道

```
速干长袖T恤 + 薄抓绒衣 + 冲锋衣 + 速干长裤 + 防水徒步鞋 + 羊毛袜 + 帽子 + 墨镜
```

- 背包里放：厚抓绒衣/轻羽绒（休息时穿）、雨衣、水、零食

### 场景3：蒂卡普湖观星（夜间）

```
速干长袖 + 厚抓绒衣 + 轻薄羽绒服 + 冲锋衣 + 长裤 + 秋裤 + 羊毛袜 + 保暖帽 + 围巾 + 手套
```

- **重点：非常冷！** 0°C左右，风大
- 可带暖宝宝贴

### 场景4：皇后镇缆车/滑板车

```
速干长袖 + 薄抓绒衣 + 冲锋衣 + 长裤 + 休闲鞋
```

- 山顶风大，带件外套

### 场景5：Onsen温泉

```
泳衣（必须带！）+ 换洗衣物
```

- 温泉提供毛巾，但建议带自己的泳衣
- 泡完温泉换干衣服

### 场景6：蓝企鹅归巢（夜间）

```
速干长袖 + 薄抓绒衣 + 冲锋衣 + 长裤 + 保暖帽 + 手套
```

- 海边风大，比市区冷5-10°C

### 场景7：Fine Dining餐厅（Rata等）

```
衬衫/薄外套 + 休闲裤 + 休闲鞋
```

- 新西兰餐厅没有严格 dress code
- 休闲正装即可，不用西装领带

## 五、行李箱建议

### 行李箱

| 类型 | 建议 | 说明 |
|------|------|------|
| **大行李箱** | 1个（28寸）| 放大部分衣物 |
| **登机箱/背包** | 1个（20寸）| 日常随身，装贵重物品 |
| **徒步背包** | 1个（20-30L）| 户外活动用 |

### 打包技巧

1. **衣物卷起来放** — 省空间，减少褶皱
2. **鞋子套塑料袋** — 防脏
3. **重物放底部** — 行李箱更稳
4. **常用物品放登机箱** — 万一托运行李延误
5. **留点空间** — 回程买手信
