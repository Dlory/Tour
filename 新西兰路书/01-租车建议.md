# 01 - 租车建议

## 一、驾照要求

### 你的情况
- **香港驾照** ✅ 新西兰承认，可直接使用
- 无需额外办理国际驾照（IDP）
- 建议同时携带**护照原件**，租车时核对身份

### 注意事项
- 驾照必须在**有效期内**
- 香港驾照上的英文姓名需与护照一致
- 部分租车公司可能要求驾照持有满**1年或2年**，预订前确认

---

## 二、车型推荐

### 推荐车型：中型SUV（Compact / Mid-size SUV）

| 车型 | 代表车款 | 适合原因 |
|------|----------|----------|
| 紧凑型SUV | Toyota RAV4, Mazda CX-5, Subaru Forester | **最推荐** — 空间够、油耗低、通过性好 |
| 中型SUV | Toyota Highlander, Ford Everest | 行李多或想更宽敞可选 |
| 轿车 | Toyota Camry, Mazda 6 | 便宜但山路/碎石路不如SUV从容 |

### 为什么推荐SUV？
- 南岛山路多、弯道多，SUV视野更好
- 库克山、格林诺奇等地可能有碎石路
- 2人+2个大行李箱，SUV空间更舒服
- 春季山区可能有残雪，SUV通过性更好

### 不建议的车型
- ❌ 小型轿车（如Toyota Yaris）— 长途山路累，行李可能放不下
- ❌ 大型房车（Campervan）— 2人用太浪费，停车不便，油耗高

## 五、保险建议

### 必买：全险（Full Insurance / Zero Excess）

| 保险类型 | 说明 | 建议 |
|----------|------|------|
| **碰撞险（CDW）** | 车辆碰撞损坏 | ✅ 必买 |
| **盗窃险（THW）** | 车辆被盗 | ✅ 必买 |
| **玻璃轮胎险** | 挡风玻璃、轮胎损坏 | ✅ 建议买，碎石路多 |
| **道路救援（RSA）** | 抛锚、爆胎救援 | ✅ 建议买 |
| **个人意外险（PAI）** | 驾乘人员医疗 | ⚪ 可选，如有旅行险可不买 |

### 重要提醒

1. **Zero Excess = 零起赔额**
   - 意思是小刮小蹭也不用赔钱
   - 强烈建议购买，新西兰修车极贵
   - 全险通常每天多 NZD 20-30

2. **信用卡保险？**
   - 部分高端信用卡自带租车保险
   - 但通常有起赔额（Excess NZD 2000-4000）
   - 且理赔麻烦，建议还是买租车公司的全险

3. ** gravel road 条款**
   - 部分保险不覆盖未铺装路面（碎石路）
   - 去格林诺奇、库克山可能经过碎石路
   - 预订时确认保险是否覆盖

## 六、预订渠道（详细操作指南）

### 第一步：比价平台（推荐先用这些比价）

#### 1. Rentalcars.com（最推荐比价）
- **网址**：https://www.rentalcars.com
- **特点**：全球最大租车比价平台，聚合各大公司价格
- **操作步骤**：
  1. 打开网站，选择语言（右上角可选中文）
  2. 输入取车地点：`Christchurch Airport (CHC)`
  3. 取车时间：`2026年9月28日 09:00`
  4. 还车时间：`2026年10月5日 15:00`
  5. 驾驶员年龄：选择你的实际年龄（25岁以上选择最多）
  6. 点击搜索，按价格/评分排序
  7. 筛选车型：选择 `SUV` 或 `Intermediate`
  8. 对比价格，记录最便宜的几家
- **注意**：显示价格通常不含保险，点进去看总价

#### 2. Booking.com
- **网址**：https://www.booking.com/cars.html
- **特点**：和租车公司直接合作，有时有会员优惠
- **操作步骤**：
  1. 登录你的Booking账号（有Genius会员更便宜）
  2. 输入基督城机场 + 日期
  3. 筛选 `SUV` 车型
  4. 对比Rentalcars的价格
- **优势**：如果你已有Booking Genius等级，可能有额外折扣

#### 3. 租租车（中文平台，适合新手）
- **网址**：https://www.zuzuche.com
- **特点**：中文界面，中文客服，有问题好沟通
- **操作步骤**：
  1. 打开网站或APP
  2. 选择 `新西兰` → `基督城`
  3. 输入日期时间
  4. 筛选SUV车型
  5. 查看报价（含中文保险说明）
- **优势**：
  - 全中文，保险条款看得懂
  - 有微信客服，随时咨询
  - 提供中文GPS租赁
- **注意**：价格可能比英文平台稍贵一点，但服务省心

  7. 支付预订（信用卡预授权，到店付或预付）

- **操作步骤**：
  1. 输入 `Christchurch Airport (CHC)`

  3. 选择 `SUV` 车型组
  4. 选择保险：`Excess Reduction` 或 `Maximum Cover`
  5. 填写信息并支付
- **优惠码**：搜索 `Avis discount code NZ 2026`

#### Budget 官网

- **特点**：通常比Avis便宜10-20%

## 八、取还车 checklist

### 取车时
- [ ] 检查车身划痕、 dent，拍照记录

- [ ] 确认无额外收费
- [ ] 索要还车凭证

## 九、新西兰自驾交通规则要点

| 规则 | 说明 |
|------|------|
| **右舵左行** | 方向盘在右边，靠道路左侧行驶 |
| **让行规则** | 右转让左转，支线让主线，环岛让右 |
| **安全带** | 全员必须系，违者罚款 NZD 150 |
| **限速** | 市区50，郊区100，学校区域40（注意标志）|
| **手机** | 开车禁止手持手机，可用蓝牙/支架 |
| **酒驾** | 极严，血液酒精 < 0.05% |
| **超车** | 虚线可超，实线不可；多在直道超 |
| **单车道桥** | 注意标志，箭头小的一方让行 |

### 推荐APP
- **Google Maps** — 导航主力，提前下载离线地图
- **Maps.me** — 离线地图备用
- **CamperMate** — 加油站、厕所、景点信息
